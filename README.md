# machine_learning_fotovoltaica
Este repositório contém o projeto da disciplina de Machine Learning voltado ao processamento de dados voltados a sistemas fotovoltaicos. O conteúdo inclui o arquivo jupyter notebook com todo o pipeline de tratamento dos dados, a apresentação em slides com o resultado e o arquivo de ambiente virtual .yml para a devida replicação das dependências. Os dados utilizados não foram incluídos por excederem o limite de armazenamento, porém o arquivo (Passo a passo para baixar os dados (interesse no projeto 2021 a 2025).txt) detalha o procedimento para baixá-los e organizá-los corretamente antes de rodar o código. Para executar o projeto, basta configurar o ambiente via terminal e apontar o diretório de dados conforme indicado na documentação interna.
